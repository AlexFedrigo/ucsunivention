Observação Importante: Independente o Sistema Operacional de Servidor (Windows ou GNU/Linux),
sempre configurar a Placa de Rede em Modo Estático (Endereço IPv4 ou IPv6 Fixo), em Desktop
(Windows ou GNU/Linux) já é recomendo utilizar endereços Dinâmicos (DHCP) em alguns casos
setamos o endereços estático para efeito de compartilhamento de recurso do Desktop (Impressora, 
Scanner, etc...)

Primeira etapa: Ingressando o Microsoft Windows Server 2008 R2 no Domínio UCS
		hostname
		ipconfig /all
		nslookup pti.intra
		ping pti.intra
		Windows Explorer
			Botão direito do mouse em cima do ícone: Computer, selecionar: Properties
				Computer name, domain, and workgroup settings
					Change settings
						Computer Name
							Computer description: Servidor Windows 2008 R2
							Change
								Member off: 
									Domain: pti.intra
									User: administrator
									Password: pti@2018
		gpresult /r
		\\ptispo01dc01
